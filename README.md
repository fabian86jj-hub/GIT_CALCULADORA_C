NOMBRE DEL PROYECTO 
"conversor de monedas.c++"

DESCRIPCION DEL PROYECTO 
"Es un proyecto universitario para los alumnos de TDS, donde realizamos una aplicacion conversora de monedas (quetzales a dolar, o euros eto)
programa el cual trabajado desde una máquina virtual (UBUNTU), con ayuda de VSC.

PREREQUISITOS (TECNOLOGIAS A INSTALAS) 
-GIT y acceso a GITHUB. 
-Oracle virtual box -Ubuntu (maquina virtual) 
-VSC (visual studio code)

PASOS PARA SU EJECUCION 
1-Abre tu usuario GIT y entra al siguiente repositorio: conversor.c 
2-Abre VSC y conecta mediante SSH a tu Maquina virtual. 
3-Ejecuta los comandos para crear html desde VSC ya conectado a ubuntu. 
4-Crea una carpeta con el nombre que desees. 
5-Crea un archivo con extensión ".c" 
6-Copia el código de la carpeta conversor.c en el repositorio y pegalo en la carpeta creada en el paso 5. 
7- este es un codigo es compilado por lo cual debes escribir el siguiente comando para  ejecutar el conversor: ./conversor
